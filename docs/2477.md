TITLE:
[App] Sorting by ❤️ in Reports Overview sorts by Title instead

LABEL:
app

STATE:
closed

BODY:
**Describe the bug**
On a users Overview page in the Reports section, if you try click the ❤️ column to sort by number of hearts, it instead sorts by Report Title.

**To Reproduce**
Steps to reproduce the behavior:
1. Go to [https://wandb.ai/stacey](https://wandb.ai/stacey)
2. Scroll to Reports section
3. Click ❤️ Column
4. See that it hasn't sorted by number of ❤️, it instead has sorted by title.

**Expected behavior**
It should show the highest ❤️ed Reports, not sort by title.
